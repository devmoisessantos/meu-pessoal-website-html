Protetor de PDF com Flask

Mais um projeto acadêmico pra trilha Desenvolvedor Web.
Ele foi desenvolvido utilizando o Framework Flask, uma biblioteca Python para construir Sites e Aplicações Web.

A proposta desenvolvida é uma aplicação web com intenção de que usuários possam fazer upload de seu arquivo PDF.
Onde o usuário coloca o seu CPF, escolhe onde deseja incluir ao PDF e então realiza o upload do mesmo.
Após esses passos o PDF seré protegido e gerado um novo PDF com o CPF informado no local escolhido.
